# src/modules

This folder contains the domain modules of the application. Follow the project structure defined in `docs/project-structure-0204.md`.

Each module should follow the canonical shape:

- domain/
- application/
- infrastructure/
- index.ts
- README.md

If a sub-folder has no files yet, it should still exist (possibly with an `index.ts` placeholder) so the structure is consistent across modules.
